# Incident #03 — File Server Unresponsive

## Date: April 2025  
## Severity: Critical
## Status: Resolved

---

## Problem
Employees could not access shared drives. File server stopped responding 
to all requests during business hours.

---

## Initial Symptoms
- Shared drives showing as disconnected on all workstations
- RDP to server timing out
- Server ping successful (server was online but unresponsive)
